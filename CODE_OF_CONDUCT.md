# Code of Conduct

## Our Pledge

KasSigner is a security-critical project. We are committed to providing a
welcoming and respectful environment for everyone, regardless of background.

## Our Standards

- Be respectful and constructive
- Focus on technical merit
- Accept constructive criticism gracefully
- Prioritize security over features
- Report vulnerabilities responsibly (see SECURITY.md)

## Enforcement

Unacceptable behavior can be reported to kassigner@proton.me.
